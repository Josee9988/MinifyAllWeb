What is MinifyAll?

MinifyAll is an original tool made by @Josee9988 (Jose Gracia) fto minify your web resources easily and fast.

The minifier supports up to 4 main languages: CSS, HTML, JSON and JSON-C. But we also allow the minimization of JavaScript via Terser.

MinifyAll was created as a VSCode, but it is now available as a terminal command and package to use in your projects and as a web (MinifyAllWeb).

The project is open source and every one can freely contribute to the improvement of the tool.

The minifier uses regex as its main tool to minify the code, it finds unused spaces, and useless code and removes it. Also, it is fully written in TypeScript!
